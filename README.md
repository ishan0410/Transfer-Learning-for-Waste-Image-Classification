# ♻️ Transfer Learning for Waste Image Classification

Built a high-performance **image classification pipeline** using **transfer learning** to classify waste images.

---

## 🚀 Highlights

- Processed & augmented **4,752 images** (resizing, padding, rotation, flipping)
- Implemented models: **ResNet50, ResNet101, EfficientNetB0, VGG16**
- Applied **dropout, L2 regularization, and early stopping** to reduce overfitting
- Evaluated using **Precision, Recall, F1-score, ROC-AUC**

---

## 🏆 Results

- **Best Model:** ResNet101  
- **F1-Score:** 0.7680  
- **ROC-AUC:** 0.9710  

---

**Python • TensorFlow • OpenCV • NumPy • Pandas • Scikit-learn • Matplotlib**
